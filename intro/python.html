Introductions to the Digital Humanities

1 Introductions to Python

1.1 Python for linguists

  • Dirk Hovy's Python for Linguists (Original)
  • Digiling https://learn.digiling.eu/
  • http://www.nltk.org/book/

Sample tasks you can start with while looking into the basic introductions:

  • make a frequency list from a document,
  • convert Wikipedia to text, see WikiExtractor,
  • extract only text from a Web page, see Trafilatura,
  • convert a tmx file to a parallel corpus,
  • add POS tags or Named entities to a text, for example, the CONLL03 model,
  • predict the genre for a text, see my Genre model

3.3 From Lena Voita's intro

Learn to think as a researcher:

  • find flaws in an approach,
  • think why/when something can help,
  • come up with ways to improve,
  • learn about previous attempts.
